# Reference anchors (clean, safe, idiomatic)

Use these as “anchors” when asserting best practices or idioms.
Prefer permalinks/versioned docs when available.

If an external reference materially influences a design/contract decision, record:
(source + retrieval date + version/tag if available) in the spec.
